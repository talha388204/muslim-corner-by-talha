# বই যোগ করার নির্দেশিকা

## নতুন বই যোগ করার ধাপ:

### ১. এই ফোল্ডারে নতুন বই এর জন্য একটি ফোল্ডার তৈরি করুন
উদাহরণ: `public/books/book-007/`

### ২. ফোল্ডারে এই ফাইলগুলো রাখুন:
```
public/books/book-007/
├── book.pdf          # মূল PDF ফাইল
└── cover.png         # বই এর কভার ইমেজ (বা .jpg)
```

### ৩. `src/data/books.ts` ফাইলে নতুন বই যোগ করুন

ফাইলটি খুলুন এবং `books` array তে নতুন object যোগ করুন:

```typescript
{
  id: "book-007",                                    // ইউনিক ID
  title: "আপনার বই এর নাম",                         // বাংলা নাম
  author: "লেখকের নাম",                             // লেখকের নাম
  description: "বই সম্পর্কে বিবরণ (ঐচ্ছিক)",      // বিস্তারিত বিবরণ
  coverUrl: "/books/book-007/cover.png",            // কভার ইমেজ এর পাথ
  pdfUrl: "/books/book-007/book.pdf",               // PDF ফাইল এর পাথ
  pages: 250,                                        // মোট পৃষ্ঠা সংখ্যা
  categories: ["ইসলামিক কর্নার"],                  // ক্যাটাগরি (একাধিক হতে পারে)
  ratingAvg: 4.5,                                    // রেটিং (০-৫)
  ratingsCount: 100,                                 // রেটিং সংখ্যা
  badge: "NEW",                                      // ব্যাজ (ঐচ্ছিক): "NEW", "BESTSELLER"
  featured: true                                     // হোমপেজে ফিচার হবে কিনা (ঐচ্ছিক)
}
```

### ৪. সম্পন্ন! 

এখন আপনার বই অ্যাপে দেখা যাবে।

## ক্যাটাগরি তালিকা:
- "সকল"
- "জনপ্রিয় বই"
- "কার্টুন"
- "সেরা ফ্রি বই"
- "সেরা বই"
- "ইসলামিক কর্নার"
- "সাহিত্য"

নতুন ক্যাটাগরি যোগ করতে `src/data/books.ts` এর `categories` array এ যোগ করুন।

## টিপস:
- কভার ইমেজ সাইজ: ৩০০×৪৫০ পিক্সেল (প্রস্তাবিত)
- ইমেজ ফরম্যাট: PNG বা JPG
- PDF ফাইল সাইজ যত ছোট রাখবেন তত ভালো (ব্যবহারকারীদের জন্য)
